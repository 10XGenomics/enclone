enclone variable inventory

This page provides an alphabetical catalog of enclone variables. We first describe the columns in the catalog, then provide the catalog itself, in several chunks.

All variables are in one of two classes:

  • lvar: a lead variable; these are not associated to a particular chain, and in clonotype tables appear at the beginning (leading), before the variables that are associated to a chain
  • cvar: a chain variable; these are associated to a particular chain.

All variables are specified at one of the following levels:

  • cell: defined for each cell
  • exact: define for each exact subclonotype
  • clono: defined for each clonotype
  • group: defined for each group of clonotypes.

In the clonotype table display, only variables whose level is cell or exact may be shown. In the PER_CELL mode, where one line is shown per cell, any variable defined at the cell level should show up for each cell, and if not, this is a flaw, and noted below. Variables defined at the exact level are not shown for each cell, as this would be redundant.
