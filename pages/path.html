Setting your path for enclone

This page explains how to edit your shell initialization file to add :~/bin to PATH. This will work in both Linux and Mac environments. If you are using a Windows computer, it will work if you use Windows Subsystem for Linux (WSL) or are using Linux through dual boot.

1. Why is this necessary?

If you do this, then every time you want to run enclone, you'll type enclone. Otherwise you'll have to type ~/bin/enclone every time, and that's inconvenient.

2. Are these instructions universal?

Not exactly. They only work if you're running the bash shell. To determine if this is the case, type echo $SHELL. If the answer is /bin/bash, then you are. Otherwise, the instructions on this page will not work. If you need help, let us know, and we will generalize these instructions to other shells as needed.

3. Now type exactly this. You only need to do it once, as it will persist "forever". 

echo "PATH=\$PATH:~/bin" >> ~/.bash_profile
source ~/.bash_profile

4. Carry on with enclone: you're all set!
